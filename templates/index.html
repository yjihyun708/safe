재난 대비 포즈 인식 훈련

Teachable Machine 기반 포즈 인식 웹 애플리케이션

1. 사용자 정보 입력

2. 핵심 서비스

Teachable Machine의 포즈 인식 기술을 활용하여 재난 상황에서의 반응 훈련을 가능하게 합니다.

3. 사용 방법과 원리

1단계: 모델 키 입력 → 2단계: 웹캠 인식 시작 → 3단계: 인식 결과 확인 및 키 입력 전송

4. 왜 이 서비스가 필요한가요?

반복적인 훈련을 위한 시뮬레이션 도구로 활용할 수 있으며, 실제 대피 훈련의 한계를 극복할 수 있습니다.

5. 사용 예시 및 사례

포즈를 취하면 해당 동작이 인식되어 실시간으로 키보드 입력으로 전송됩니다.
